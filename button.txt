0: set bằng 0
1: set bằng 1
2: toggle 1->0 và 0->1
3: teleport

[[0, (1,2)],[1, (3,1)]] : set ô (1,2) là 0 và ô (3,1) là 1
[[0, (1,2)],[0, (1,3)]] : set ô (1,2) là 0 và ô (1,3) là 0
[[1, (1,2)],[1, (1,3)]] : set ô (1,2) là 1 và ô (1,3) là 1
[[2,(1,2)],[2,(1,3)]] : là switch ô (1,2) và (1,3) (0->1 và 1->0) 
[(1,2)],(2,4)] : là teleport block1 ở vị trí (1,2) và block2 ơ vị trí (2,4)
